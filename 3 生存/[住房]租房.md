**如何租房**

```zsh
# 找中介
特点:费钱,省事,资源多,选择多
流程:租房网站找中介->明确需求,约看房

# 找房东
费事,省钱,资源少,选择少
流程:小区广告牌,宣传栏->找房东约看房
```



**租在哪里**

```zsh
# 近公司,近地铁,别太偏
打开租房软件->定位公司位置->查看3-5公里地铁站附近小区->提出偏僻小区->明确最优租赁目标
```



**租房模式**

```zsh
# 合租
合租前提：E人优先、对他人包容性高、无洁癖
优点：省钱、热闹、多交朋友、不孤单、互相照顾
缺点：隐私减少、互相打扰、物品使用、公用区域卫生

# 整租
比如:一室一厅,二室一厅,三室两厅,独立loft
整租人群：社恐I人、有洁癖、喜欢独处
优点：完全隐私、高度自由、区域独享、不受干扰、安全性高
缺点：成本较高、孤独感强、无人帮助
```

**租房避坑**

```zsh
# 新装放,串串房
看着很新,刚装修完,打开衣柜有强烈的胶水味,床垫膜还没撕掉
看着配套齐全,崭新,杂牌
5级能效空调,缩小版马桶,拼色窗帘和沙发,桌子柜子密度板
看上去没有生活痕迹
不是房东本人签合同
价格不符合它的位置,要价特别低
闻到甲醛味赶快走,小心得白血病

# 电器,电表,水电费
电器:能耗越低越好
电表:在自家是自接的电表,在楼道里是国家电表;可以在国家电网,微信,支付宝等官方渠道缴费是国家电表;房东代缴,通过小程序缴纳是自接电表
水电费:商水商电>民水民电

# 个人隐私
一层:隐私不好,蚊虫多,噪音多,采光差,暴雨天被水淹,

# 采光
注意房屋是否是在全阴面,通风不好,容易发霉

# 噪音
电器运行的声音,左右邻居隔音效果,外面的隔音效果

# 看房
白天晚上都要看,白天看采光,晚上看噪音

# 楼道卫生
楼道不干净,蟑螂蚂蚁少不了

# 室内气味
卫生间是否反味

# 家用电器
空调,冰箱,洗衣机,热水器是否都有,能否使用,能耗如何
电器的卫生情况如何,空调是发霉,冰箱是否丑了,热水器流出的水是否有水垢
电灯是否好用,出现坏的,要求房东提前做好更换

# 室内插座
插座是否足够
```



**验房签房**

```zsh
# 做好证明,保住押金
打开手机,从进门验到出门
把能试的都试一下,滴漏,下水管,洗衣机排水管顺不顺畅,有没有反臭
门窗能不能打开,有没有声音,门把手是否松动
衣柜厨具有没有异味
水表,电表,燃气表的数值全部录进去
屋内有不想要的东西,问房东能否丢弃或搬走
全都验好后才能签合同

# 合同
明确:房租,物业费,水电费分别由谁来交,怎么交,是季付还是月付
有没有其他隐形的费用
日常维护费用的归属
确认合同内转租及费用:合同期内是否能转租
合同期内房东违约条款
退押金的标准需要明确
```
