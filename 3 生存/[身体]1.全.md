**医疗**

```zsh
# 共识
生命第一(用尽一切手段让病人活下来,活着是最基本的)
疾病是与人终生相伴的(没有完美的基因,有些疾病对身体有好处:训练免疫系统)
医疗的本质是支持生命的自我修复(一切医疗都是给自我修复赢得时间,给自我修复起到缓冲作用)(如果自我修复功能丧失了,那么医疗也没用了)


# 修复
细胞不断分裂补充受损和死亡的细胞

# 代偿
细胞加快工作,代替受损的机体工作


# 身体系统

身体每个营养素,微量元素的最佳基准线(这样的身体状况最好)
终极操作:基于最佳基准线,增加一些东西.减少一些东西
最次:也不要低于生命线,低于就死



# 症状和疾病


症状:病的外在反应,身体的自我防护
疾病:病的成因
观察症状,找准病因
针对性的解决病的原因,相对的缓解症状
有些慢性病是没有症状的,这意味着当发现时就里死亡不远了
所以需要定期体检


# 医疗决策
医生:提出假设,收集证据,验证假设
患者:知情,选择,寻求支持
我真的需要做这个检查和治疗吗?
这个检查和治疗有什么风险或不足,会有什么副作用?
还有更简单安全的选择吗?
如果我什么都不做会怎么样?

患者和医生是同盟,对抗疾病
任何问题第一时间问主治医生


```
