**大脑**

```zsh
# 三元脑
爬行脑:自动驾驶
哺乳脑:快决策
人类脑:慢决策

# 神经元
刻意练习:有针对性,不断重复,训练神经元(把知识,技能长在脑子里)
神经可塑性:你可以按照自己的意愿,成为你想成为的人

# 爬行脑
# 脑干
睡意,警报,心率,呼吸,消化,体温

#小脑
保持平衡,条件反射,身体协调

# 哺乳脑
边缘系统:长期记忆,情绪管理,性唤起,嗅觉(扣带回:负面想象,焦虑)

# 杏仁核(爬行脑,人类脑,哺乳脑三大脑区交汇处)
第一反应:恐惧,兴奋,愤怒,战或逃
基底神经节:操作技巧,习惯养成,上瘾,奖赏(容易被商家控制)
下丘脑:出汗,体温,口渴,血压,心率,饥饿,性,颤抖,养育本能,生物钟

# 人类脑
# 额叶(发育晚,发育慢)
前额叶皮质:计划,分析,判断,专注,社交,自控
镜像神经元:运动,模仿,社交(学习,语言,洞察)

# 顶叶
宏观调控,处理与环境的关系
整合信息,空间想象,协调动作,数学
突然性,天然性:顿悟

# 枕叶
视觉

# 颞叶
观察,理解,识别,记忆

# 启发
同时唤醒越多的脑区越好,同一件事,运动更多的感官
训练呼吸:贯穿三元脑
重大事件的第一反应(杏仁核):只能身临其境的训练
嗅觉和情绪与性唤起有关
人类脑>哺乳脑>爬行脑(可控程度),用人类脑去训练其他脑区
不断整合信息,整合想象,可能会出现顿悟
```
