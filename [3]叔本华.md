### 作为意志和表象的世界

```zsh
# 表象
任何能够进入到意识中的东西
表象世界包含:主体,客体

# 身体
特殊客体:内外结合

# 充分根据律
理解世界的内嵌程序,规定了表象世界里所有事物,都必须以某种必然的方式互相连接
实践,空间,因果律等都是内嵌程序
意识到的一切都会被充分根据律自动处理

# 物自体
滤镜之外的真实世界

# 意志
意志是叔本华认为的物自体
意志:盲目的没有特定目的,永远不会满足的生命冲动,或是一种力量,它是驱动所有东西存在的那个最原始的渴望本身,没有理由的要存在和表现出来(死亡驱力)
纯粹生命力:在石头身上表现为重力,在植物身上表现为生长,在人身上表现为欲望

# 意志的客体化
世间的一切都是意志的不同形式的显现

# 艺术
理念:意志不同层级的客体化
艺术是理念的摹本,纯粹主体:感受理念而忘却痛苦与无聊
从低到高:建筑->悲剧
音为特殊的艺术:直接体验意志本身

# 美学->伦理学
艺术哪怕是音乐都只是暂时缓解
而如果想要解决,需要从关照意志,转向彻底否认意志

同情与禁欲:
肯定生命意志:我一定要活下去,我想要的东西一定要得到,他人不重要(个体幻觉)
同情:其他人跟我是同一的,都是受苦的,生命意志的不同显现而已
万物都是同一个在受苦的意志

寂灭:生命意志被彻底消灭后的平静(简直胡扯)

肯定的无，对应精神现象学的感性确定性，否定的无，对应知性的分解破坏力，叔本华说把意志取消了，表象也就消失了，然后人们怕面对这种虚无，就会逃进涅槃，他自己则勇敢地选择面对，收获宁静。这种低智商前反思的隔岸观火式自恋，完全没有意识到，意志（符号学秩序）解体后，表象不是消失，而是变成群魔乱舞，主体不是宁静，而是首先陷入混乱和狂暴，因为主体本身就是符号学秩序设立自身，廓清场域后的结构性剩余。

这种低级的默认主体可以在世界之夜中幸免于难的前观念论的立场，充其量是19世纪的幻想文学。

这种宁静，归根结底是对面对苦难和匮乏的人民和被压迫者的呻吟和惊慌失措的嘲笑，是慷他人之慨的恶毒和愚昧。

恬不知耻。
```
