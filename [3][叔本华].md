**作为意志和表象的世界**

```zsh
# 表象
任何能够进入到意识中的东西
表象世界包含:主体,客体

# 身体
特殊客体:内外结合

# 充分根据律
理解世界的内嵌程序,规定了表象世界里所有事物,都必须以某种必然的方式互相连接
实践,空间,因果律等都是内嵌程序
意识到的一切都会被充分根据律自动处理

# 物自体
滤镜之外的真实世界

# 意志
意志是叔本华认为的物自体
意志:盲目的没有特定目的,永远不会满足的生命冲动,或是一种力量,它是驱动所有东西存在的那个最原始的渴望本身,没有理由的要存在和表现出来(死亡驱力)
纯粹生命力:在石头身上表现为重力,在植物身上表现为生长,在人身上表现为欲望

# 意志的客体化
世间的一切都是意志的不同形式的显现
```
