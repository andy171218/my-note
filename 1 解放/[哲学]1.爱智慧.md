**形而上学**

```zsh
# 哲学的目的
哲学要达到一种自治的统治

# 如何实现
第一步:放弃预设的秩序(支配现实的规则),逃到形而上学(尝试解释不可解释的东西,看上去神秘莫测)
形而上学是一种失败学,在现实的神权王权面前没有办法,逃到精神世界中去寻求一点生机
为什么不能沉溺在形而上学中,因为形而上学是一种自我安慰,他无法继续前进,而一切停下来的东西都是敌人
所以形而上学是一种哲学的失败

# 现实和理想
现实不等于真实,所以现实主义是什么?现实主义是在预设的规则秩序之中按照规则玩游戏,而理想主义就是去探索秩序的漏洞与可能性
想象(现实),象征(理想),真实(漏洞)(三个是联动的,如果一个动了,另一个没动,那就是停止了)
象征秩序发展了,想象图示也必须跟着发展,不然就是假的

# 错误的方式
由形而上学推导出伦理学,然后支配生活
```



**真正基础**

```zsh
哲学并非简简单单的我想过我想要的生活
想要的生活必须得到充分的论证

智慧:不动的推动者,哲学:对智慧的追求
把不断追求智慧看成是自己的生活方式,在追逐过程中不断的产生对于一切的结论,那自己的生命去执行这些你认为善的结论
终极问题:智慧是不是我爱的对象

# 真正基础
爱情,爱智慧,首先你的会爱,受过爱的折磨,疑问:我到底爱不爱,我到底爱的是什么
当你突然爱上一个人,这个人原本平凡的一切突然变得不平凡,这就是形而上世界
哲学:哲学家对于世界的爱,把整个世界当作恋人,将这个世界的真相献给这个世界
哲学作为一种对世界的爱,将世界的家:形而上学作为礼物献给这个世界,抚慰孤独的世界
这个世界本身最应存在的方式,推导出我最应的生存方式,绝对信念

# 爱智慧
爱智慧,获得智慧但不占有,把智慧拟人化:伴侣,一直陪伴着你的人生,与智慧相伴,保持一种激动的宁静
问题:只能感受到人间的痛苦(创伤,不一致性),而无法感受到世界本身的痛苦(创伤,不一致性)

哲学是爱智慧,不代表哲学是爱真相的,哲学是憎恨逃避真相的

# 古典哲学->现代哲学
尼采:不应当爱智慧,应当爱真相,爱命运,重估一切价值
不是通过逻辑得到的结论而是通过生命的挣扎

# 尼采之后
关注视角本身,爱意味着什么:为什么原先非常庸俗会一下子变得十分美好
原先的无限美好的爱变成了一种相爱相杀,与命运缠斗但是依然爱着它
高贵的被动性:任凭命运摆布,因为爱命运
颠倒的基础:被爱,爱命运就是接受自己被爱的状态
```
