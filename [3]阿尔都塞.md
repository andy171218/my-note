# 意识形态vs理论
意识形态不是对现实的真实全面的反应,而是扭曲的不完整的,会掩盖粉饰或合理化社会中存在的根本矛盾,尤其是阶级对立和剥削关系
意识形态:阶级利益的观念体系,顽固,统治作用:维护统治阶级的合法性,用幻想统治;处于想象界
理论:超阶级性,自我革新,实践验证,科学的理论可以揭露意识形态;处于符号界
意识形态的阶级性:意识形态工具会被统治阶级利用
反对人道主义的马克思(早期:1844年经济学哲学手稿),支持科学的马克思

# 意识形态
意识形态:表述体系,包含形象,神话,观念所构成的环境,一出生便沉浸其中
意识形态的功能不在于认识真理,而是操作行为(无意识)
未经反思的被当成理所当然的假设都是意识形态,对现实的想象关系,误解,前科学,表象
意识形态会暗示现实,同好给予幻术,对于现实的扭曲.人想要这样,这样理解世界对人有某种好处和利益
意识形态是人对生活的意愿(保守,顺从,改良,革命)
统治阶级自己体验并相信自己创造出来的意识形态,他们并不是只把意识形态当成工具,而是自己也活在这些意识形态中,只不过这些意识形态更有利于他们而已
被统治阶级也会生产自己的意识形态,但是这些意识形态也是位居从属地位,会受到统治阶级意识形态的影响,连反抗的工具本身也来自对方(摇滚)
统治阶级统治被通知阶级,统治阶级的意识形态统治被统治阶级的意识形态
大众文化产品:	对真实生存条件的想象关系的表征
意识形态都是会去掩盖回避现实,只不过有的文化产品聪明,有的笨拙
意识形态国家机器:宗教,教育,家庭,法律,政治,工会,媒体,文化(文学,艺术,体育)(内部充满了矛盾和斗争)(意识形态战争)
镇压性国家机器(镇压,强制,暴力):维护统治阶级统治,政府,行政机关,军队,警察,监狱,法庭
教育和家庭是现代最主要的意识形态国家机器:对于那些只能从事底层体力的劳动者,用教育使他们顺从,容易管理;对于成为管理者的人,培养他们的人际交往能力,教他们如何对工人发号施令;对于可能会进入镇压行国家机器的人,教他们无条件服从统治阶级和对被统治者发号施令的能力;对于未来会成为政治领袖的人,教授修辞术和说服技巧;对于教师,僧侣,知识分子,训练他们如何有效的影响他人的意识形态,为主流意识形态打掩护,生产主流意识形态
意识形态具有物质性:不是观念决定行为;而是行为决定观念,教会这个物质机器,教会制定的一整套物质实践和物质仪式
意识形态是在各种意识形态国家机器中所处的位置,以及参与其中的实践活动而导致的结果
只要存在阶级社会,意识形态就不可避免;比起掩盖矛盾,还是不断的去揭露矛盾吧
意识形态对于主体的塑造:
询唤:呼唤;权威性的审问,质询
意识形态不断的把生物学的人形动物,呼唤成主体,一个看似独立的思想和行为的中心
出生前:赋予的名字,对于性别的预设,家庭的期待
预设大他者:在大他者的呼唤中成为主体,主体就是对于上帝的臣服者
先把你抬升为一个自由的人,然后让你"自愿"的臣服,全靠自己做出臣服的表现和行为
读书,读什么书不重要:重要的是在什么地方读,在什么环境和背景读,用什么方法读
意识形态通过让我们相信自己是自由的主体,来确保我们处于臣服的状态,从而维护了现存的剥削性的社会关系

# 经济基础决定上层建筑
经济基础:生产力,生产关系
上层建筑:法律,文学,哲学,艺术,政治
经济是基础但不是一切,上层建筑由反作用力甚至在,某些时候起到决定性的作用
上层建筑对于经济基础有相对自主性和特殊效能
经济基础的改变并不会直接的改变上层建筑,上层建筑有自己相对独立的发展轨迹,自己的领地和历史
经济基础起到决定作用而不是起到作用的只有经济基础,任何结果都是经济基础和上层建筑的复杂互动所产生的

# 过度决定
一个结果是由多个不同的原因造成的,这些元音来自不同的层次
结果就是整个结构运动过程本身

# 不平衡发展
在不同时期,起到最大作用的层次不同
不同层次的权重,随着时代的变化而变化
每个层次内部的发展也是不同的,比如文学中的小说,戏剧发展不同

# 实践(内在性外在化)
不同层次就是不同的特定的实践
经济实践,意识形态实践,理论实践,整个社会就是不同实践的结合体
原料:被转变的原来的对象
生产资料:物质工具,技术,知识,方法,组织,手段
产品:转换后的成果
政治实践:现有的社会关系,社会矛盾,诉求,不满;政党组织,工会,法律条文,议会,警察,军队等国家机器,政治策略,演说技巧,动员能力,选举,立法,示威,革命;新的社会关系,新的权力结构,新的法律政策
理论实践:已有的概念和方法;逻辑方法;新的知识和理论

# 结构主义
能指链的滑动:一个符号的意义取决于这个符号与这个系统中的其他符号的位置上的差异

# 难题性
思想被一个框架所制约,决定了哪些问题可以被问出来,哪些概念可以被思考,哪些问题问不出来,哪些概念无法被思考
这个框架不仅决定了答案,往往还决定了什么问题应该被提出
在认知一切(书籍,电影,对话...),要找到其依赖的底层框架
早期人道主义的马克思,依赖的框架是费尔巴哈的人道主义
晚期科学的马克思,依赖的框架是历史唯物主义

# 认识论断裂
符号学系统的发展是革命性的断裂而不是渐进改良的
人道主义只是一种意识形态
马克思对于黑格尔体系的超越也是断裂性的,不是简简单的首尾倒置就好了的
人一开始就是社会的人

# 症状阅读(口误,矛盾,遗漏,沉默.空白,bug)
文本的沉默比言说表达的更多
挖掘文本的潜意识
症状:文本试图在某一个难题性框架中去思考和表达,但是这个框架本身由局限性,当文本触碰到框架的边界,试图处理超出框架的部分,就会产生说不通,矛盾或避而不谈的地方(阅读符号秩序的不一致性)
例子:一份报告是否只谈好处不谈对某些群体的坏处,是否选择性的呈现出有利的部分,通过这些可以读出这份报告试图掩盖和回避的某些社会现实,利益冲突
一般阅读:找中心思想,幻想符号秩序的整全性,完美性,没有意识符号秩序的断裂

# 劳动

剩余价值:劳动所产生的价值超过了劳动力

# 知识
知识是生产出来的,不是观察出来的
原材料:旧的理论,意识形态
生产资料:难题性(新的分析框架:历史唯物主义)
产品:新的理论
所谓的科学即现有的最先进的难题性

# 文学
批判:
1. 文学内部有一个内在的核心意义或秘密,读者要做的是挖掘出文本内部作者想要表达的真理,这种想法太天真了,忽略了意义的复杂性和文本内部的矛盾性
2. 将作品完全等同于作者的意图,认为作品就是作者本人思想,情感,经验的表达,读者要做的是猜测作者到底想说什么
文学不是神秘的天才式的创造,而是一个特定社会历史条件下进行的生产过程的产物
艺术实践(意识形态实践):对意识形态进行加工就是艺术,社会流行观念,刻板印象,价值判断,个人经验历史记忆,个人宗教信仰;可利用的文学体裁,诗歌,小说,戏剧,叙事,修辞,风格,象征,出版,发行,流通,审查,想象力(被特定的社会条件,意识形态,可利用的文学工具所决定的);艺术作品
由于文学作品是由这些内在矛盾的意识形态所构成的,所以生产出来的文学作品其本质必然是不同的,异质的,文学作品内部会充满多样性甚至是各种意义的冲突,而不会是和谐完美的整体
文学作品会有张力,意义的缺席,无法解决的矛盾,文学作品是去中心化的,它的统一性并不在作品自身内部找到,无法从作品内部找到一个唯一的统治性的核心意义
文学作品的统一性或者造成其内部矛盾的原因在作品外部的那些产生了意识形态原料的充满矛盾的社会历史现实

批判的任务:
解释为什么这部作品是这个样子,分析作品内部的不一致性,如何通过其功情节人物,叙事风格无意识的批判了哪些构成了它的意识形态原材料
当散乱的意识形态要在文学这种具体有结构的形式中被表达出来时,就像是一堆杂乱的东西要放到一个固定的箱子中,他们内在的矛盾反而会更容易暴露出来,文学形式可以逼迫意识形态现原形

作品的无意识就是历史:作品避而不谈,内部矛盾所指向的东西就是主流意识形态所遮蔽的真实历史状态

形式变换不改变内容

# 哲学归根到底是阶级斗争在理论领域的延续,哲学是理论的阶级斗争
没有中立客观的理论,没有中间地带,它们总是服务于两种根本立场中的一个:压迫与被压迫(维护现有秩序或推翻现有秩序)
无产阶级立场:历史唯物主义,划清科学和意识形态的界限,保护推动科学的发展(阶级斗争是历史的动力)
资产阶级立场:经验主义,唯心主义,人道主义,混淆科学和意识形态,阻碍科学发展,将科学拉入意识形态泥沼(人是历史的创造者:掩盖了阶级斗争)
积极的介入文化领域的斗争,要去争夺对经典文本和当代文化产品解释权,为被压迫者的解放服务

# 艺术,科学,意识形态
科学(历史唯物主义科学):提供对现实,包括对意识形态的认识,概念性的外部的分析:在某种特定的社会形态下,某种意识形态,它的本质,它如何产生,它有什么社会功能,怎么服务于统治阶级
艺术:看到,感觉到,察觉到,沉浸式体验
不存在普遍的艺术,一切艺术都要放到具体的条件下去看
创造新的观众而不是取悦现有观众
关注点永远是艺术的结构

# 来日方长
幻觉也是事实,没有内在性,都是外在性
记忆不是对过去的客观录影,而是过去透过现在主体的欲望,情感,和无意识被不断的重构内心 
作为自主的、有自由意志的、作为一切经验和行动中心的个人或主体，其实是一种意识形态的建构，是意识形态运作的基本范畴
主体不重要,人不重要,重要的是符号秩序能否表象,物质能否生成新的符号秩序,这三者的运作就是实体,实体能否继续运作下去
实践对于任何意识都具有优先性

同一个现实,对于不同阶级的人来说看到的是不一样的结构,意识形态就是这样起作用的
意识形态和文本不断的重塑与加工对方

主导的,过时的,残余的,新兴的(领导全,反领导权)

总有一些社会存在和意识领域是主导模式所不能允许存在的
感觉结构:实践意识领先于官方意识,实际体验优先于能够清晰表达他们的语言和文化形式;不是无意识或沉默,是社会性的物质性的感觉和思想,但还处于胚胎期,无法用现有的体系所囊括
