# 公共交通



## 飞机

```zsh
# 选机场
有的城市是没有机场的
地图软件搜索查看附近的机场

# 买机票
12306 & 航旅纵横 & 携程
航班:价格,时间,航司
价格:越低越好
时间:飞机时间+提前2个小时去机场+延后30-50分钟离开机场
航司:

传统航空
票价:相对高
行李政策:
随身大行李加上一件小行李
随身行李的尺寸限制大约为20厘米×40厘米×55厘米，可以理解为20寸行李箱，重量通常限制10公斤之间

免费额度的行李托运
头等舱：通常会有40公斤
商务舱：通常为30公斤
经济舱：20公斤

餐饮服务:
一般会提供免费的飞机餐和饮品供你选择,随时免费获取饮用水

廉价航空
票价:相对低

行李政策:
一件随身行李加上一个小包
随身行李通常限制在大约40厘米×30厘米×20厘米范围内，且般不超过7公斤
无免费额度的行李托运

餐饮服务:
不包含免费餐食和饮料，如需用餐则要额外购买

传统航空:
·中国国际航空公司（AirChina）：中国唯一的载国旗飞行的民用航空公司，也是星空联盟成员之一。
·中国东方航空公司（ChinaEasternAirlines）：拥有广泛的国内外航线网络，是天合联盟成员。
·中国南方航空公司（ChinaSouthernAirlines）：中国最大的航空公司，同样也是天合联盟成员。
·海南航空公司（HainanAirlines）：以其优质的服务著称，也是寰宇一家的成员。
·深圳航空公司（ShenzhenAirlines）：属于国航系，提供高质量的服务。
·四川l航空公司（SichuanAirlines）：地方性航空公司，以成都为主要基地，覆盖国内及部分国际航线。

廉价航空:
·春秋航空（SpringAirlines）：中国首家由旅行社起家的廉价航空公司，以提供低成本服务闻名。
·九元航空（NineYuanAirlines）：以广州为基地，提供国内航线服务，是吉祥航空旗下的廉价航空品牌。
·中国联合航空（ChinaUnitedAirlines）：前身为军用航空机组转型而来，现已成为一家重要的廉价航空公司。
·西部航空（WestAir）：以重庆为基地，主要运营国内航线，也有部分国际航线。
·成都航空（ChengduAirlines）：总部位于成都，主要经营国内航线。
·幸福航空（JoyAir）：标志性的“五福临门”LOGO让人印象深刻，主要运营国内旅游城市的航线。
·首都航空（CapitalAirlines）：虽然是一家综合性航空公司，但部分业务已转型为低成本模式。

价格:搜索价格+机建费+燃油费

# 去机场
提前2个小时到达机场
确认到机场的时间和出发时间
确认航站楼

# 值机
选座位:线上选座位
1. 靠后:颠簸感越重,下飞机晚
2. 靠近机翼:视线受阻,噪音大
3. 靠近过道号上厕所

办理托运:当天去机场办理托运
看机票的航空公司,看指示牌对应的位置
身份证交给工作人员,如果没有选座位:想要偏向什么样的座位
托运:称重,安检扫描
注意有没有免费额度,没有或超重要交钱
最好不要托运,托运也最好不要托运贵重物品,不要超重
取登机牌和托运牌

禁止携带物品:先收拾行李,再查行李中有哪些禁止携带,如果有不得不带又不能走飞机的就走快递

# 登机安检
禁止:液体,易燃易爆物品
检查手续:身份证,登机牌
正式安检区:大件行李,小件行李,随身携带物品
前往登机区等待
登机:飞机起飞前30分钟,早上飞机有位置放行李

# 起飞途中降落
等到平稳可以打开手机,没有网,飞行模式
提供:飞机餐,饮料,毛毯(有限)
在巡航高度才能上厕所,下降就会停用

# 下飞机,取行李
行李转盘,拿托运行李,检查
行李柜台:行李损坏或丢失
一公斤100元,所以不要托运贵重物品
```



## 高铁

```zsh
# 购票:窗口购票,自助购票机,线上APP购票:12306

# G/D/C,复兴号
考虑出发时间:首选复兴号或者G/C开头
考虑性价比就选D开头
静:代表有静音车厢
兑:代表可以通过积分兑换

# 进站,安检
临时身份证:温馨服务
30分钟前到达车站
先整理行李,之后在查有没有禁止携带的物品

# 候车&检票
确认检票口位置

# 站台候车
确认车次

# 乘车
红色:当站有人
绿色:当站没人,下一站也没人
黄色:当站没人,下一站有人
最好不要在高铁上点吃的,太贵了
充电插座,有的车型没有
提供免费饮用水
卫生间:红色代表有人

# 换乘&出站
换乘:换乘通道,问工作人员
遗失物品:12306填写信息
```
