# 反本质主义
只有在本质主义的视角下,才有本质与现象的二分
尼采不关注本质,他是反本质主义,它只关注现象,所以才会有后面的结构主义
在本质主义中:存在要么作为名词,作为事物的基础;要么作为系动词,第一运动

# 生存,过程,成为
become,不断的超越,变化

# 坠落
如果走成功了,智慧将会变为强力(权力)
成功的尼采:希特勒
走成功的人:斯大林

# 思想史
爱智慧(古希腊)->现代哲学(过渡:黑格尔)->爱命运(尼采)->现象学运动(反动:重新爱智慧)->纠正一(海德格尔:现象学黑格尔化,走钢丝成功的尼采,纠正一失败)->纠正二(法国神学现象学传统->结构主义:重新爱命运->符号学(结构是符号的结构:指称关系->符号间产生的效果->历史本身)->后结构主义运动)
命运:他者的智慧
非理性包含着理性

# 三个步骤
直接爱智慧(理想,一致性,看不见不一致性,柏拉图传统,形而上学传统,没有历史,认为神是完美的,只要赞美爱者神即可)->神学(伪堕落)->观念论(idealism:观念,形而上世界,太一生病了,这些病就是历史,自然,现实;而太一的诊断学就是辩证法:推敲它得了什么病,会发展成什么样,所有的观念论哲学都是一本精神病理学的诊断手册,患者是世界本身)->从爱抵达智慧的过渡(堕落的过程,哲学的历史化)->爱他者的智慧(被智慧所爱,被迫的爱上智慧)

# 逻辑学
逻辑学->自然哲学->精神哲学
太一生病(神一开始就疯了)->形成稳固的自然(症状)->炎症(发烧)
神&自然&精神:人&呼吸&火焰
主体性的夜晚可以烧掉一切:烧掉自然,烧掉神,回归到比神更开端的东西
神从安宁的状态->发疯开始不停的说,受不了了想要遗忘,把这些记忆甩到外面去变成自然->外在性的自然(不受控,自动运行)发展:人类,文明,精神,伦理,宗教,艺术,哲学->用哲学治愈生病的神:帮助神回忆起他发疯的全过程,然后用否定性烧掉一切,回到开端(安宁)

但是这样真的能够治好神吗,让神回忆起一切就够了吗?

你能体会到神在痛苦,在挣扎吗?

黑格尔主义:最后把自我取消了,把责任取消了,所以还是失败的
