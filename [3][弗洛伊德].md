```zsh
# 性

# 记忆
记忆使人成形
在心理生活中,事物一经形成便不会消亡,一切往事都可以设法让它重现。
那些我们以为忘记的或是因为太痛苦,太羞耻不符合社会规范而被我们刻意或不自觉地排除在意义之外的记忆、欲望、冲动它们并没有消失,而是被压抑到了无意识这个心理区域里。但它们仍然具有能量会以各种伪装的形式:梦里，在口误中，在无法解释的情绪或行为模式里，不断地试图重现，影响我们。

# 无意识
如果说精神是我们要探索的地图，那无意是就是这张地图上最广阔最神秘，也是蕴藏最多宝藏或者说最多地雷的区域。
我们意识层面的思想、情感和行为，很大程度上是由我们无法直接意识到的无意是力量所驱动和决定的,凡是心理的事物皆首先应看作是无意识的存在,意识不过是心理事物的一种特性，而且绝非恒常不变的特性。

储藏库:不被意识所接纳,被意识遗忘的东西,最原始的本能欲望和冲动，尤其是性和攻击性，因而拾其未被满足的需求和往来，还有那些因为太过痛苦或创伤而被压抑下去的童年记忆。
心灵垃圾桶:从来不会清空

动力系统:无意识并不是一个被动的储仓式，而是一个充满了能量、欲望和冲突的活跃区域。它和意识与前意识,处于一种持续不断的互动，甚至是对抗的关系(被禁止的欲望、创伤记忆，它们会不断的想要冲破束缚，进入到意识层面)

内心的很多挣扎，焦虑，甚至心理症状，都可以理解为是这种意识与无意识之间持续冲突的结果。

# 神经症
没有器质性病变,但是病人就是感觉很难受

# 压抑
意识的守门员:阻止那些不被意识所接受的，可能引发焦虑或痛苦的无意识材料，进入到意识的大门里,它需要耗费心理能量来维持这种压制。

# 解释
精神分析首先是一种阅读的理论(精神分析式的阅读)
现实是被扭曲了的,被扭曲了的才是现实
现实即被扭曲了的真实
将精神分析的方法运用到艺术,文学,社会行为

# 梦的解析
梦是通往无意识的康庄大道
显梦是经过伪装的潜在欲望隐梦的实现
口误,笔误,丢三落四,遗忘("故意"遗忘)(无意识冲突或欲望泄露)
诙谐和无意识的关系:笑话好笑是因为巧妙地绕过了我们内心的检查机制，让我们可以短暂地以伪装的形式，释放一些平时被压抑的攻击性或性的冲动
无意识动机:动机往往是你意识不到的
意义背后的复杂性和动力
```
