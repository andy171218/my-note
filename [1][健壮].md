### 医疗

```zsh
# -------------- 看病流程 -------------- #
# 准备:身份证,手机,医保卡(支付宝:电子医保)
# 去医院
# 挂号:收集患者的基础信息,确定看病归属科室,选择医生和看病时间,交钱
普通号:经验较少
专家号:经验较多
# 挂号方法
现场挂号:窗口挂号,自助挂号机
线上挂号:支付宝微信小程序,医院官网/app,第三方诊疗平台
# 拿着挂号条去对应科室
签到排号
# 看医生
直达病灶:开药方,缴费取药,出医院
有点大病:开住院单,住院,办理住院手续+缴费,护士站(手环,床位,床头卡),病房(医生办公室:诊断,看病,签手续),办理出院手续
继续检查:开检查单,检查科室...等结果

# 共识
生命第一(用尽一切手段让病人活下来,活着是最基本的)
疾病是与人终生相伴的(没有完美的基因,有些疾病对身体有好处:训练免疫系统)
医疗的本质是支持生命的自我修复(一切医疗都是给自我修复赢得时间,给自我修复起到缓冲作用)(如果自我修复功能丧失了,那么医疗也没用了)


# 修复
细胞不断分裂补充受损和死亡的细胞

# 代偿
细胞加快工作,代替受损的机体工作


# 身体系统
身体每个营养素,微量元素的最佳基准线(这样的身体状况最好)
终极操作:基于最佳基准线,增加一些东西.减少一些东西
最次:也不要低于生命线,低于就死

# 症状和疾病
症状:病的外在反应,身体的自我防护
疾病:病的成因
观察症状,找准病因
针对性的解决病的原因,相对的缓解症状
有些慢性病是没有症状的,这意味着当发现时就里死亡不远了
所以需要定期体检


# 医疗决策
医生:提出假设,收集证据,验证假设
患者:知情,选择,寻求支持
我真的需要做这个检查和治疗吗?
这个检查和治疗有什么风险或不足,会有什么副作用?
还有更简单安全的选择吗?
如果我什么都不做会怎么样?

患者和医生是同盟,对抗疾病
任何问题第一时间问主治医生

# -------------- 预防 -------------- #
# 避免高危因素
预防,治疗慢性感染:注射疫苗
预防>治疗(疫苗>药>手术)

# 避免高危行为
抽烟,喝酒,熬夜

# 定期体检(对疾病进行早期筛查)
肺癌,乳腺癌,宫颈癌,直肠癌

# 不要破坏和细菌的内共生(不用抗生素,别太干净)
保护身体不受致病菌的侵害
合成身体所需物质
锻炼免疫系统

# 增强免疫力(最底层的,自愈就靠免疫力)
疫苗
不严重感染性疾病可以锻炼免疫系统

# 体检
体检的目的是为了预防疾病

# 急救
当我说出一种伤病，或拍照片，你需要对我提供急救指导（直接说出行动指令）和治疗建议（比如：去医院挂什么科；中医，西医两方面的建议）
```
