**系统论**

```zsh
# 系统,结构,循环,飞轮,习惯...
要能识别出系统,然后能够对比不同的系统,然后扬弃系统,让系统越来越好
系统就是自动化,把内部性外部化
下判断:一个系统是会毁灭,还是会进一步发展(这个系统:收益>成本or成本>收益)
永远盯着长期,盯着大趋势,而不是短期(物理是光鲜亮丽还是跌入谷底)

# 三大组件
系统:相互联系的实体构成的有运作规律的整体
系统=要素+关系+作用
要素:+-,关系:+-,作用:快慢

滞后效应:缓慢的增加,缓慢的减少
滞后效应的前提是长期坚持,如果中途断了,那么也没有效果
所以好系统和坏系统本身就是会带来滞后效应,慢慢的变好和慢慢的变差
不要高估短期能发生的变化,不要低估长期能发生的变化

# 好系统三大特征
适应力:超量恢复,挖掘潜能
# 储蓄池
储蓄池是一直有流入也有流出的
开源节流可以增大储蓄池,如果有源头,那么储蓄池永远不会枯竭,如果没有源头,那么储蓄池总有流完的一天,就像人总会死一样(人的生命总有一天会消亡,但是人活这一生,只要是不断探索源头就没问题)
储蓄池大,抗风险性越大

自组织:一个系统不依靠靠外部指令运转,而是按照某种内在规则,自动自发地形成有序结构的一种现象
层次性:子系统

# 杠杆解
构造出一个因果关系网
然后找它的漏洞,抓住漏洞

# 建立仪式
做第一个动作,剩下的自动完成
最重要的是第一个抓手,必须简单

# 构造新系统
首先识别现有系统,找到其漏洞(最关键,最重要的一点),利用漏洞改善系统
大目标:重塑系统,分解整个系统,然后在单位结构上建立仪式,最后每个仪式形成习惯,这样新的系统就改善好了
```

```zsh
# 熵增定律
# 熵和状态数成正比
熵越小:状态数越少,越秩序
熵越大:状态数越多,越混乱

封闭系统,无外力做功,熵会越来越大
因为最大的封闭系统是宇宙,所以总系统的熵一定会越来越大
但是允许局部熵减,生命系统

# 人生启示
如果随波逐流,人会越来越混乱而不是有序
混乱是常态,有序是需要可以营造的
坚持自我控制练习,坚持精粹结构
保持开放的态度
坚持做功:努力,操作而不是放弃,不要随波逐流,不要被熵牵着鼻子走
端正:可以集中精神
请不断的做出感知和选择:正确的感知和正确的选择可以变得有序
增加正信息:让自己变有序(知识),减少负信息:让自己变混乱(不要胡思乱想)
因上努力,果上随缘:内外(内:有序,外:混乱)
```
