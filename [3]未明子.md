## 1 形而下学

```zsh
# 位格
场=1:实在论(Realism)一一认为世界是融贯的、均匀的(不仅空间融贯，时间亦是融贯，两百年前是怎样的两百年后还是怎样)秩序的一个“场”，而人是这个“场”里面的一个东西（你甚至不会把你的出生、死亡、创伤性的事件当成是这个场的断裂），觉得你死了之后这个世界还继续存在。

# 代表人物
其典型思维模式是追问“What is...?”（什么是……），预设任何符号都对应场域中的一个确定对象，并相信存在一个能对所有事物进行分类和定位的整体性符号系统。古希腊喜剧作家阿里斯托芬在《云》中塑造的苏格拉底被视为此种思维的代表。

# 主要内容
“形而下学”（1- Prephysics）界定为一种天真的、前反思的哲学实在论。其核心特征是认为世界是一个融贯、均匀且稳定的秩序场域，而人只是这个场域中的一个存在者，不会将个体的出生、死亡等创伤性事件视为场域本身的断裂。

存在论基础：这种思维继承了亚里士多德式的存在论，即将“存在”思为“存在者”，致力于对存在者整体进行分类和定性，但因此遗忘了“存在”本身与“存在者”之间的差异（即海德格尔所批判的“遗忘存在”）。

场域论的局限：形而下学的核心在于对“存在者整体”的把握，但它未能反思“存在者整体”与其所处的“场域”或“秩序系统”之间的张力关系。它默认场域是均匀且不言自明的。

# 通向形而上学的关键
要超越这种天真状态，需要认识到 “一切”（存在者整体）与“一切之所在”（场域本身）是分离的，并思考场域本身是否是一个存在者？以及“无”是否具有本体论地位？这种思考将引导哲学从对“是什么”的追问（形而下学），转向对“何为一切？”（What is total?）的追问，从而进入形而上学（2- Metaphysics） 的领域。这一跨越需要一种激进的主体性（如通过复仇主义的总体化体验）作为中介来揭示其中的张力。

# 总体=总体化与整体
总体=总体化:一个动力学中心点来总体化(主体性)(有否认、缝合的机制,但凡它看到的剩余，它都会回性地说“这本来就是我的一部分”，所以总体只能是一个不断总体化的运动)(他的国族民族概念可以根据共同体的利益不停修改的)
整体:无脑、无动力学中心点的(松散的前现代国家)
哲学:认定总体与总体化有差异(观念论,形而上学)
辩证唯物主义:认定总体和总体化是一样的,并且随时随地对于哲学的姿态保持一种警惕

# 形而上学vs观念论
形而上学vs观念论:形而上学没有意识到主体性的存在(它没有意识到这个过程当中，有一种不同于作为存在者的场域，背景的“无”的另外一种“无”，这种“无”它根本不能被填充、不能显在、不能有任何一种存在者在它上面)

也就是说从形而上学到观念论，就要意识到场域本身是需要一个动力学机制所结构的（符号系统是有生产性的，它需要动力学机制生产出这么一个空白的场，同时这个动力学机制本身也永远是一个乏的场。虚无不再是什么都没有，而是它永远缺东西，别人永远无法占据它，它像虚数一样不能在实数轴上作为0被表达）

# 总结
简而言之，形而下学代表了一种将世界视为既定、有序且可完全被符号系统所分类和把握的前反思哲学姿态。克服这种姿态，需要意识到世界秩序本身并非天然融贯，而是充满了裂缝与张力，并由此开启真正的哲学（形而上学）思考。
```

一种人人都可进行的思维训练，即从生活中抽离出概念和情境，像玩魔方一样进行排列、组合和推演，以此锻炼思维、疗愈自我，再返回生活。这种哲学旨在成为个人生活的顾问型AI”，而非主宰事业的“主人”

哲学活动是专题性的抽离，但它内置了一种返回生活的机制。它不像科学探索那样追求无限的外部扩张，而是在内在思维达到某种完备性后，自觉地回归生活实践

倡导一种“非正式的专题性”哲学：从生活（如爱情）中抽取出要素（男性、女性、欲望、恐惧），像玩思维游戏一样进行交叉组合、逻辑推理，不讲概念史，只讲语法和修辞，敢于转变词性、打比方。
通过这种游戏，生活中的概念会失去其吓人的力量，变得熟悉和可操作，从而实现自我疗愈。

这种思维游戏能形成一种“思维习惯”或“内在小朋友”，它会自动玩耍，并将其玩出的有趣观点引诱”你，为你提供建议，这就是“顾问性”的来源。哲学的本质是幼稚的、游戏的，其力量来自趣味性。
将哲学从“苦大仇深的思辨”转变为“快乐的内在游戏”。持续的游戏会在潜意识中形成一种新的思维模式，这个模式会自动工作，在你需要时提供新颖的、解放性的视角，这就是你个人的“哲学AI顾问

# 酷刑
强调哲学的象征性和语言性。同时也指出其危险性：当沉迷于思维游戏而脱离生活时，哲学不再是解放的工具，而会成为自我折磨的刑具。

# 哲学训练作为打造内在AI
大量的哲学阅读和训练就像在训练一个人工智能（AI）模型。这个“内在AI”会成为你的一个“顾问”，会自动对你的情绪、所见社会现象进行分析和批判。
它说明哲学学习的目的是重塑我们的思维习惯，在大脑中形成一个能自动进行深度处理、批判性思考的“第二人格”或“顾问”。这与单纯学习科学知识（只会用技术话语侵占生活）完全不同。

# 哲学与科学在生活中的不同表现
哲学训练旨在培养一种融入生活的智慧；而科学思维则是一种专业性的、需要转换才能应用于生活的工具。直接套用会显得格格不入

# 哲学与事业:主人还是顾问
哲学可以成为事业的“主人”（导师、支配者），但这是残忍、蛮横的（如斯大林主义）。讲座者更倾向让哲学成为事业的顾问”（（辅佐者）。

人类的伴侣一一有限、庸俗但不可或缺。它源于婴儿期的无力感（对周遭符号的被动接收与尝试支配），也显现于临终前的“走马灯”（对人生可能性的重新设想）。

# 总结
哲学是“作为可能性的生活的探索”，是“生活的基础”，呼吁大家善于进行这种思考，打破生活的锁。
哲学是一门生活的艺术和技术，其终极目的是通过思维上的自由获取生活中的自由。
